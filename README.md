# Sistema de Cadastro de Clientes

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Flask](https://img.shields.io/badge/Flask-2.3.3-green.svg)
![SQLite](https://img.shields.io/badge/SQLite-3-lightgrey.svg)

Sistema completo de cadastro de clientes desenvolvido com Flask e SQLite, com interface responsiva e moderna.

## 📋 Funcionalidades

- ✅ Cadastro de clientes com validação de email único
- ✅ Listagem de todos os clientes
- ✅ Edição de dados dos clientes
- ✅ Exclusão de registros
- ✅ Busca em tempo real na tabela
- ✅ Interface responsiva (mobile/desktop)
- ✅ Mensagens de feedback (flash messages)
- ✅ Design moderno com gradientes

## 🚀 Tecnologias Utilizadas

- **Backend:** Python + Flask
- **Banco de Dados:** SQLite
- **Frontend:** HTML5, CSS3, JavaScript
- **Ferramentas:** Git, VS Code
